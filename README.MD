Сделать поле для ввода текста (textarea) с ограниченым количеством символов достпуным для ввода в строку и лимитированым количеством строк.
Поле ввода 25 (длина строки) на 8 (количество строк);
При вводе должны соблюдаться определённые условия:
1. Длина текста в каждой строке не должна привишать маскимальную.
2. Количество введённых строк не должно превышать максимальное.
3. Если ввод (каретка) доходит до конца строки выполняеться переход на следующую автоматически.
4. Если вводить текст на строке где уже есть текст то он должен смещаться в право, а слова которые не влазят переноситься на следующую строку, делай сдвиг на ней.
5. Всё что не влазит в последнюю строку должно удаляться.
6. При достижении конца поеследней строки указателем, ввод текста должен блокироваться.
7. При реализации логики переходов и движений необходимо сохранять позицию ввода (картеки) на сколько это возможно.
8. При вставке скопированого текста требования више должны соблюдаться.

Рушение нужно прислать в видет одного html файла с JS кодом в нём или ссылкой не гитхаб или битбакет с открытым доступом.
Можно использовать jQuery. Если не все требования учтены, описать какие.
